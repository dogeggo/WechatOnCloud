# 开发提示词

## 基本约定

- 始终使用简体中文沟通；环境按 Windows 11、PowerShell、UTF-8 处理。
- 需要库/API 文档、安装、配置或代码生成时，优先用 Context7 查询官方资料。
- 新功能先划清边界：HTTP、业务编排、AI 调用、JSON/DTO、SQL、JSX 展示不得混进同一类或函数。
- 不需要任何兼容老数据的逻辑，不要写兜底逻辑，报错直接走前端错误弹窗；
- 复用两处以上的规则、默认值、选项归一化、序列化、节点/布局/引用逻辑，抽到领域 `Manager`、`Factory`、`Serializer` 或前端 `utils`。
- 改完清理死代码、未用 import、重复 helper；
- 编码任务完成只需要编译通过即可，不需要运行验证。
